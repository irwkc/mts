# Сценарий демо-видео (5–8 минут)

1. Запуск: `docker compose up -d --build`, открыть `http://localhost:3000`, войти в Open WebUI.
2. Выбрать модель **`gpthub-auto`**, коротко объяснить, что это автоматический режим.
3. Текстовый вопрос по общей теме — ответ LLM.
4. Загрузить скриншот или картинку — показать ответ vision-модели.
5. Написать: «Нарисуй логотип кофейни в минималистичном стиле» — показать результат генерации изображения.
6. Вставить ссылку на статью и спросить по содержанию — показать парсинг URL.
7. Написать: «Найди в интернете последние новости про …» — показать веб-поиск в контексте.
8. Вставить длинный фрагмент текста (или «документ»), задать вопрос по нему — RAG.
9. Сказать фразу «Запомни: мой любимый язык программирования — Python»; в новом сообщении спросить «какой язык я люблю?» — долговременная память.
10. Переключить на **ручной** выбор конкретной модели из списка и отправить сообщение — ручной режим.

При монтаже добавить субтитры с указанием: единый тред, MWS GPT, режимы авто/ручной.
